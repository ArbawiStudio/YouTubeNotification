### How Get Channel ID

```js
module.exports = {
    channel: "", // Notification Text/News Channel
    messageTemplate: "Hello @everyone, **{author}** just now uploaded a video **{title}**!\n{url}", // Notification Message
    channel_id: "" // YouTube Channel ID
}
```
